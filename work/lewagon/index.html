Le Wagon Lille

Le Wagon is a training to become a Web developer in Ruby on Rails.

I'm currently a teacher for Le Wagon Lille, since April 2016, it's a fullStack program of 2 months to become a web developer.

Le Wagon is a coding school founded in 2013 in Paris, France, it's mission is to bring technical skills to creative people.

The 9-week coding bootcamp gives the ability to build a prototype of any idea.No technical background needed, just motivation.

There is 1190+ students already graduated from Le Wagon.
